# OKF / Google Documentation Support

This directory contains documentation support files for OKF-style and Google Code Assistant workflows.

## Purpose

- Provide a dedicated location for knowledge-transfer documents aligned with OKF and Google documentation conventions.
- Store templates and guidance that can be used by Google Code Assistant or other implementation agents.

## Contents

- `README.md` — overview of the OKF/Google doc support directory.
- `google_code_assistant_template.md` — template instructions for generating code using Google Code Assistant.

## Usage

1. Use `google_code_assistant_template.md` when you want to prompt Google Code Assistant directly.
2. Keep this folder stable as a shared repository contract for external doc-generation workflows.
